# Shopping Cart

A responsive website that allows users to view items and add / remove items from cart.
They can also filter items based on its class.

Built with:
<ul>
  <li>React</li>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ul>
